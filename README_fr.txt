Bonjour!

Ce projet ci est déstiné d'être un outil éducational pour les élèves en école
primaire pour pratiquer des opérations arithmétiques avec des fractions. Ce
project est open source, et est déstiné d'être une expérience éducationale.

Ce projet était dévéloppé avec LibGDX, duquel vous pouvez apprendre plus
ici: https://libgdx.badlogicgames.com
